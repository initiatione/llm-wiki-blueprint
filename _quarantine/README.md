# Quarantine

This is the archive review zone for notes with weak signal, weak connections, or unclear long-term value.

Notes moved here are not automatically deleted. They can be:

- revived if they regain relevance
- merged into stronger canonical notes
- compressed into summaries
- retired with a tombstone record

## Rules

- Prefer quarantine over immediate deletion.
- Keep rare or hard-earned notes out of auto-delete flows.
- Review this folder periodically.
- If value can be extracted, merge it into stronger canonical notes before removal.
