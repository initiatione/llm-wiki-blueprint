# Draft Workspace

This is the default staging area for agent output.

Drafts are the publication buffer between `_raw/` and Canonical Knowledge.

## Rules

- Drafts are not the default factual source for downstream agents.
- New pages should enter here before promotion when risk is non-trivial.
- Large updates, Cross-domain Synthesis notes, and contradictory changes should enter here first.
- Review state lives in `_meta/review/`.

## Flow

`_raw/` -> ingest -> `_drafts/` -> review -> approve/reject -> canonical folders
