# wiki_ext Example

This directory is a public example of an outer enhancement layer.

It is not meant to be your live private enhancement repo.

Use it as:

- a reference
- a starter layout
- a pattern for keeping extra workflow logic outside the base workflow repo

## Included Areas

- `review-flow/`
- `wrappers/`
- `governance/`
- `migrations/`
- `templates/`
