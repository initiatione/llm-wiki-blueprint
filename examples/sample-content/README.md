# Sample Content

This directory contains a minimal example of how content can move through the blueprint workflow.

The goal is not to provide real private knowledge.
The goal is to show:

- what a raw note looks like
- what a draft looks like
- what review state looks like
- what a promoted canonical note looks like

## Example flow

```text
sample-content/_raw -> sample-content/_drafts -> sample-content/_meta/review -> sample-content/references
```
