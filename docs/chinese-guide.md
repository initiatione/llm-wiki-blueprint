# 中文导览

这份文档是本开源蓝图仓库的中文入口。

它的作用是：

**让中文使用者不需要先读英文，也能快速理解这套 llm-wiki 蓝图。**

## 你先记住一句话

这套蓝图的核心不是“让 AI 自动乱写知识”，而是：

**让 AI 负责整理和起草，让用户保留最终发布判断。**

## 建议阅读顺序

1. `README.zh-CN.md`
2. `_meta/guides/当前 llm-wiki 框架说明（中文）.md`
3. `_meta/guides/智能体工作说明（中文）.md`
4. `_meta/guides/英文文件中文索引.md`

## 你会看到什么

- 推荐目录结构
- 草稿与审稿流程
- Canonical Knowledge 的分类方式
- review / quarantine 的意义
- 智能体在系统中分别做什么

## 重要说明

英文原文保留，是为了让英文智能体和英文工作流更容易消费。  
中文文档是为人类导览服务的，不替代英文规则层。
