# Security

## Scope

This repository is a public blueprint repository.

It should not contain:

- API keys
- private sessions
- raw private source documents
- local runtime logs
- user secrets

## If you find sensitive content

Please report it privately rather than opening a public issue.

## Guidance for adopters

When using this blueprint for a real vault:

- keep your live knowledge vault private unless intentionally publishing it
- keep runtime state out of version control
- keep execution-agent configuration and secrets outside the public repository
