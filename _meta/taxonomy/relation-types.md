# Relation Types

Use typed relations when connecting notes.

- `uses`: a note relies on another method or concept
- `implements`: a note describes an implementation of another note
- `extends`: a note adds scope or capability to another note
- `evaluates`: a note measures or tests another note
- `depends-on`: a note requires another note to make sense or function
- `contrasts-with`: a note differs in a meaningful way
- `applies-to`: a note is used in another domain or context
- `contradicts`: a note challenges an existing claim
- `supersedes`: a note replaces an older note
