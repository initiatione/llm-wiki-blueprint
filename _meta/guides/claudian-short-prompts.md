# Claudian Short Prompts

Use these short prompts when you want the system to do the right thing with minimal overhead.

## New Content

```text
请按 llm-wiki 流程整理 _raw 里最新导入的内容。
```

## New Content With Images

```text
请按 llm-wiki 流程整理 _raw 里最新导入的内容，优先结合图片内容。
```

## Rework a Draft

```text
这版先 reject，请按反馈重做一版 draft。
```

## Approve a Draft

```text
这版可以 approve，请按合适目录归档。
```

## Conservative Mode

```text
先保守处理，不要直接写正式知识区。
```
