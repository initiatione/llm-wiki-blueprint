# Review Quick Decisions

Use only three decisions most of the time.

## 1. Through

Use when:

- the draft is clear
- the recommendation looks right
- the risk level is low or acceptable
- the content is bounded and sourced

Example:

```text
这版可以 approve，请按合适目录归档。
```

## 2. Rework

Use when:

- the draft is too broad
- the target folder seems wrong
- the summary is weak
- the content over-generalizes
- image content or source handling is incomplete

Example:

```text
这版先 reject，请按反馈重做一版 draft。
```

## 3. Leave It for Later

Use when:

- the content is not urgent
- you are unsure about value
- it should stay conservative for now

Example:

```text
先保守处理，继续留在 _drafts，不要进入正式知识区。
```

## Rule of Thumb

- `low` -> most often through, unless the target folder is clearly wrong
- `medium` -> glance and decide
- `high` -> read carefully before approving
