# Raw Intake Policy

`_raw/` is the default intake folder for newly captured material.

## Use `_raw/` for

- web clips
- pasted rough notes
- imported markdown
- screenshots and image-derived notes
- source fragments that have not yet been distilled

## Do not use `_raw/` for

- approved canonical notes
- review metadata
- long-term storage of already processed notes

## Workflow

1. capture into `_raw/`
2. ingest into `_drafts/`
3. review
4. approve into canonical folders

## Operational Note

If a capture comes from Obsidian Clipper or a browser clipper workflow, the preferred target is `_raw/`.
