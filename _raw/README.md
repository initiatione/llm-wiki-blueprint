# Raw Intake

Use `_raw/` as the single landing zone for new captured material.

This includes:

- web clips
- pasted notes
- rough markdown notes
- screenshots or image-derived notes
- imported source summaries

## Rules

- New material should land here first, not directly in Canonical Knowledge.
- Obsidian Clipper should target this folder.
- Treat everything here as unreviewed input.
- Promote useful content into `_drafts/` or canonical folders through the normal workflow.

## Suggested Use

- use `templates/source-note.md` for manual source capture
- use `templates/web-clip-note.md` for clip-oriented capture
- keep one note per source where possible
- include URLs and dates when available
