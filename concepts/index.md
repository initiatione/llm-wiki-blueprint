# Concept Layer Index

The concept layer exists to stabilize vocabulary and reusable abstractions across the knowledge base.

## Use this layer when

- a term appears in multiple domains
- a definition needs a single source of truth
- domain notes would otherwise redefine the same idea repeatedly

## Do not use this layer when

- the note is mostly implementation detail
- the note is specific to one project context
- the note is better represented as a Cross-domain Synthesis note
